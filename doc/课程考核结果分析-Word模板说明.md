# 课程考核结果分析表 Word 模板说明

## 模板形式

用于 **课程目标达成情况评价报告** 中「六、课程考核结果分析」表格的 poi-tl 占位符及表格要求。

### 1. 占位符

- **标签名**：`course_result_analysis_table`
- **模板中写法**：`{course_result_analysis_table}`

### 2. 表格要求

| 项目 | 说明 |
|------|------|
| 位置 | 占位符 **必须** 位于 Word **表格内部** |
| 结构 | 表格至少 **1 行 1 列**，占位符放在该单元格内 |
| 渲染后 | 原表格所有行会被删除，由策略 **动态生成** 表头 + 数据行（含合并单元格） |

### 3. 操作步骤

1. 在 Word 模板中插入一个 **1×1 表格**（或任意行列表格，仅用于定位）。
2. 在 **唯一单元格**（或你希望放置该表的任一单元格）内输入：`{course_result_analysis_table}`。
3. 保存模板并上传为流程用 Word 模板。
4. 导出报告时，该表格会被替换为完整的「课程考核结果分析」表（表头 + 数据行）。

### 4. 生成表结构说明

- **表头**：2 行。第 1 行为「课程目标」、各考核类型（如期末考核 60%、平时考核 30%、实验考核 10%）、「课程目标达成值」；第 2 行为「评价依据」「平均得分/目标分值」「达成值」等子列。列数由 `config` 决定，**无需** 在模板中预先画好。
- **数据行**：每行一个课程目标，对应 `rows` 中的条目。
- **合并**：表头存在列合并（每组考核 3 列）、行合并（「课程目标」「课程目标达成值」跨 2 行），均由策略自动处理。

### 5. 数据来源

- `config`、`rows` 由 `FlowTaskServiceImpl` 从 `getCourseAbilityLinkData` 与 `buildCourseAbilityLinkTableRows` 提供，并组装为 `course_result_analysis_table` 传入模板。
- 模板中 **只需** 正确放置 `{course_result_analysis_table}`，无需配置其它字段。
